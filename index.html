What is Nano Banana 2?

It combines Pro-grade visual quality with Flash-tier speed, delivering near-perfect text rendering, subject consistency across 5 characters and 14 objects, native 4K resolution, and real-time search grounding — all at up to 40% lower API cost than Nano Banana Pro.

Core Capabilities

Text Rendering

Near-perfect legible text. Supports marketing mockups, infographics, menus, diagrams. In-image localization across multiple languages. Wrap exact text in quotes for precision.

Subject Consistency

Lock 5 characters + 14 objects across multiple generations. Face-locking with reference images. Frame-to-frame consistency for animation prep.

Semantic Editing

Natural language edits without manual masking. Model identifies target objects and applies changes while preserving everything else. Multi-turn conversational editing.

Reference-Based Generation

Upload 6-14 reference images for style transfer. Mix objects + character refs. Last uploaded reference determines output aspect ratio.

Search Grounding

Google Search + Image Search integration for real-time accurate rendering. Generate infographics with live data. Verify facts during generation.

Transparent Assets

Green screen generation with auto background removal. Broadcast-grade quality via FFmpeg pipeline. Auto-crop padding. Perfect for game sprites and overlays.

Thinking Mode

Adjustable reasoning levels (minimal/high). Physics + intent reasoning for complex scenes. Better handling of multi-layered, multi-constraint prompts.

Dimensional Translation

Convert 2D sketches to photorealistic 3D renders. Maintain proportions and style. Style transfer while preserving subject identity.

Prompting Best Practices

Use Natural Language

Write conversational descriptions, not comma-separated tags. Describe the scene as if telling someone what to photograph.

BAD: "cyberpunk samurai, neon, rain, 4k" GOOD: "A lone samurai standing in a rain-drenched cyberpunk alley, neon signs reflecting off wet pavement, shot from a low angle with a 35mm lens"

Edit, Don't Regenerate

If an image is 80% right, refine it conversationally instead of starting over. The model preserves context across turns.

"Change the lighting to golden hour" "Move the character slightly to the left" "Make the background more blurry"

Specify Context & Intent

Tell the model the purpose. "For a Michelin-starred restaurant Instagram post" gives it the artistic judgment to make the right decisions.

"A plated dessert, photographed for a fine dining restaurant's social media — macro lens, shallow depth of field, dark moody background with a single spotlight"

Material & Texture Details

Be explicit about physical properties for maximum realism.

"Brushed aluminum with micro-scratches" "Velvet with subtle sheen under side lighting" "Aged leather with patina and worn edges"

Text in Images

Wrap exact text in quotes. Specify font characteristics. Add "high contrast for readability" and "typography-focused."

"A coffee shop menu board with the text 'COLD BREW $5' in bold sans-serif, white chalk on matte black, clearly legible, typography-focused"

Semantic Negative Prompting

Describe what you want, not what to avoid. The model responds better to positive direction.

BAD: "No cars, no people, no modern buildings" GOOD: "An empty medieval cobblestone plaza with timber-frame buildings and hanging flower baskets, morning mist"

Camera & Lens Language

Photography terms dramatically improve output quality. Specify focal length, aperture, film stock.

"Shot on 85mm f/1.8, shallow DOF" "Kodak Portra 400 film grain" "Wide-angle 24mm, leading lines" "Hasselblad medium format quality"

Resolution Strategy

Match resolution to your use case. Don't waste money on 4K for a social media thumbnail.

512px → Quick concepts, pixel art 1K → Social media, web graphics 2K → Print marketing, typography 4K → Large format, texture study
nano-banana-2-skill CLI

Open-source CLI + Claude Code plugin from kingbootoshi/nano-banana-2-skill (MIT license)

Installation
git clone https://github.com/kingbootoshi/nano-banana-2-skill.git
cd nano-banana-2-skill
bun install
bun link

# Set API key
echo "GEMINI_API_KEY=your_key_here" > ~/.nano-banana/.env
Basic Generation
nano-banana "A serene mountain lake at sunrise, mist rising"
With Resolution & Aspect Ratio
nano-banana "Product hero shot of headphones on marble" -s 4K -a 16:9 -o headphones-hero
Transparent Asset (Green Screen)
nano-banana "Fantasy warrior character sprite, full body" -t -s 2K
With Reference Images
nano-banana "Same character in a different pose" -r ref1.jpg -r ref2.jpg -r ref3.jpg
Pro Model for Maximum Quality
nano-banana "Complex fantasy battle scene with 5 characters" -m pro -s 4K
Check Spending
nano-banana --costs
